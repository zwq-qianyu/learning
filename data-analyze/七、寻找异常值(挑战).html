七、寻找异常值(挑战)

寻找异常值

说明

在前面的实验中我们学习过四分位数。四分位数，是指将一组数据值按大小排序后分成四等分，每一部分包含 1/4 的数据，这种划分的分割点就是四分位数。其中第1部分和第2部分的分割点称为第1分位数 Q_1Q1​, 也被称为第25百分位数（上四分位），第3部分和第4部分的分割点称为第3四分位数 Q_3Q3​，也被称为第75百分位数（下四分位）。而第二部分和第三部分的分割点是第2四分数，也就是中位数。其中四分位距 IQRIQR 是指第三四分位数和第一分四分位数的差，也就是 IQR = Q_3 - Q_1IQR=Q3​−Q1​。

所有小于 Q_1 - 1.5IQRQ1​−1.5IQR 或 大于 Q_3 + 1.5IQRQ3​+1.5IQR的数据项被称为异常值。

在本挑战中，你需要在 ~/Code/outlier.pyPython 文件中编写一个查找上文中定义的异常值的函数 find_outerlier，find_outlier 函数接受一个列表参数，代表数据序列。函数将查找列表中的所有异常值，并将所有异常值作为列表返回。

我的解法

# -*- coding:utf-8 -*-

import numpy as np

def find_outlier(data):

    outlier = []

    # sort the data

    print(data)

    sorted_data = np.sort(data)

    print(sorted_data)

    q1 = sorted_data[int(0.25*len(sorted_data))]

    q3 = sorted_data[int(0.75*len(sorted_data))]



    iqr = q3 - q1

    for one in sorted_data:

        if (one < q1-1.5*iqr) or (one > q3+1.5*iqr):

            outlier.append(one)

    return outlier



data=[-10,4,6,9,3,12,9,3,6,3,7,37]



print(find_outlier(data))

import numpy as np

def find_outlier(data):

 outlier = []

 Q1 = np.percentile(data,25)

 Q3 = np.percentile(data,75)

 IQR = Q3 - Q1

 outlier_step = 1.5*IQR

 for col in data:

  if (col< Q1- outlier_step or col> Q3 +outlier_step):

   outlier.append(col)

 return outlier

这里主要学习了numpy的percentile()方法，第一个参数传入列表，第二个参数传入需要计算的百分位。
